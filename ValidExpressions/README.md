# Mathematical Expression

Check whether a given mathematical expression is valid.

**Input Format:** Mathematical Expression
**Output Format:** VALID or INVALID

## Example 1

```
Input: (a+b)(c+d+e)
Output: VALID
```

## Example 2

```
Input: (a+b))
Output: INVALID

```

## Example 3

```
Input: (ab+)
Output: INVALID

```
