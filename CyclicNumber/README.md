# Cyclic number verification

Verify the given number is cyclic or not

**Input Format:** num1 num2
**Output Format:** Yes or No

## Example 1

```
Input: 12345 45123
Output: Yes
```

## Example 2

```
Input: 12345 54123
Output: No

```

## Example 3

```
Input: 123456789 678912345
Output: Yes

```

**Constrains:** 1<=range<=9999999999
