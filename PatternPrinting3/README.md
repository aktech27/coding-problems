# Pattern Printing 3

Given number of rows and a character to print, print the character in the cross pattern(ie Diagnol from left-right and from right-left)

## Example 1

```
Input:
  Rows: 5
  String: *
Output:

*       *
  *   *
    *
  *   *
*       *

```

## Example 2

```
Input:
  Rows: 6
  String: #
Output:

#         #
  #     #
    # #
    # #
  #     #
#         #

```
