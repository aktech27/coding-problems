# Sum of two numbers in the given base

Add two numbers in the given base without converting into base

**Input Format:** Get two numbers and base
**Output Format:** Display the sum

## Example 1

```
Number 1: 1010
Number 2: 11001
Base: 2

Output:
100011

```

## Example 2

```
Number 1: 123
Number 2: 13
Base: 4

Output:
102

```
