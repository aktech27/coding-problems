# Pattern Printing 1

Given a string of length, convert it into cross character or cross string(ie Diagnol from left-right and from right-left)

## Example 1

```
Input: CROSS
Output:

C       S
  R   S
    O
  R   S
C       S

```

## Example 2

```
Input: AHAMED
Output:

A         D
  H     E
    A M
    A M
  H     E
A         D

```
