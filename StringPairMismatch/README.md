# Substring Detection

Print the pair which are mismatched in two strings.

**Input Format:** String1 and String2
**Output Format:** Pairs that are mismatched

## Example 1

```
Input: abcdefgh abdfhjfb
Output: c,d  d,f  e,h  f,j  g,f  h,b
```

## Example 2

```
Input: hjdhjjduk hjdhjjdua
Output: k,a

```
