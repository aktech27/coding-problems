# Pattern Printing 2

Given number of rows, print the number in the cross pattern(ie Diagnol from left-right and from right-left)

## Example 1

```
Input: 5
Output:

1       5
  2   4
    3
  2   4
1       5

```

## Example 2

```
Input: 6
Output:

1         6
  2     5
    3 4
    3 4
  2     5
1         6

```
